# 🚀 KwremNet - Ağ Araç Seti

**KwremNet**, ağ yönetimi ve analizi için geliştirilmiş modern bir araç setidir.

## 🛠 Özellikler
* Ping Testi
* Port Tarayıcı
* DNS Sorgulama
* GeoIP Bilgisi
* Ağ Trafiği İzleme
* Traceroute

## 📦 Kurulum
1. `pip install -r requirements.txt` komutunu çalıştırın.
2. `python kwremnet.py` ile başlatın.

## ⚡ Derleme
Projeyi `.exe` yapmak için `build.bat` dosyasına çift tıklamanız yeterlidir. Çıktı `dist` klasöründe oluşacaktır.
